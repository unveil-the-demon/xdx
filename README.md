# XDX
## declaration 

首先声明，公司的工作氛围确实是我呆过所有公司里最好的，至少研发这边是这样，不管是文总、小波总还是海洋总的团队，大家都很融洽，一起debug、一起做好产品，领导也不事儿逼。
曾经幻想过在这里的美好前景，也给一些同事朋友布过道，希望他们不要怪我。
决定入职象帝先之前，也了解过这家的背景，私以为是最有希望成为国内GPU领跑者的。然而技术总归是技术，研发大佬们终究还是不懂人性..
